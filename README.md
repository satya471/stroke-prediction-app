#🧠 Stroke Prediction App
This is a Machine Learning web application that predicts the risk of stroke based on patient health data.
---
## 🚀 Features
- Predicts stroke risk using ML model
- Handles imbalanced dataset using SMOTE
- Interactive web interface using Streamlit
- Real-time prediction based on user input
  ---
  
## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- Imbalanced-learn (SMOTE)
  ---
## 📊 Dataset
Healthcare Stroke Dataset (real-world dataset)
---
